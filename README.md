# 🚀 CampusGigs – Job Portal Platform

CampusGigs is a full-stack web application that enables users to explore part-time job opportunities, apply seamlessly, and manage applications through a user-friendly interface.

---

## 🧠 Overview

This project demonstrates a complete **MERN stack implementation** with secure authentication, scalable backend architecture, and responsive frontend design.

It is designed to simulate a real-world job portal system with focus on:
- Clean API design  
- Authentication & authorization  
- Structured backend architecture  
- Responsive UI/UX  

---

## 🛠 Tech Stack

**Frontend**
- React.js
- Bootstrap

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Other Tools**
- JWT (Authentication)
- Axios (API integration)

---

## ✨ Features

- 🔐 User authentication with JWT  
- 👤 Role-based access (users/admin)  
- 📄 Job listing and application system  
- 📊 Application tracking  
- 🔄 RESTful API integration  
- 📱 Responsive UI for multiple devices  

---

## ⚙️ Architecture

- Modular backend structure using **controllers, middleware, and routes**  
- REST APIs for communication between frontend and backend  
- MongoDB for scalable data storage  
- State management on frontend for efficient UI updates
